Building URLs with url_for()

Flask provides url_for() to generate URLs dynamically instead of hard-coding paths. This makes your application safer and easier to maintain.

Why url_for() matters
  • Automatically updates URLs if routes change
  • Safely encodes special characters
  • Reduces broken links
  • Works in Python code and Jinja2 templates

url_for() in Python

from flask import url_for

url_for('home')
url_for('user_profile', username='Alice')
url_for('show_post', post_id=42)

url_for() in Templates

<a href="{{ url_for('home') }}">Home</a>
<a href="{{ url_for('user_profile', username='Alice') }}">Alice</a>
