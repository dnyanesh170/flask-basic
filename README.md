SETUP INSTRUCTIONS

1. Clone repository
   git clone https://github.com/your-username/FLASK_BASIC.git
   cd FLASK_BASIC

2. Create virtual environment
   python -m venv venv

3. Activate virtual environment

   Windows PowerShell:
   venv\Scripts\Activate.ps1

   Windows CMD:
   venv\Scripts\activate

   Mac / Linux:
   source venv/bin/activate

4. Install Flask
   pip install flask
