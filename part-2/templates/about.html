Using render_template()

The HTML content you are viewing is returned from the backend using the render_template() function. 

@app.route("/about")
def about():
    return render_template("about.html")

Template Rendering Basics

Flask uses Jinja2 as its template engine. At this stage, the page is static, but templates allow much more powerful features such as:

  • Sending variables from Python to HTML
  • Displaying lists using loops
  • Applying conditions inside templates
  • Building reusable layouts with base templates

As this project grows, templates will help separate application logic from presentation.
