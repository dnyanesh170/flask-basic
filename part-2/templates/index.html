Welcome to Flask Template Rendering

This page is served from an HTML file stored inside the templates directory using Flask.

How this page is delivered
  1. Your browser sends a request to the server
  2. Flask matches the URL with a route decorator
  3. The linked Python function is executed
  4. The index.html template is loaded
  5. The rendered HTML is returned to the browser

Benefits of Using Templates

  • Clear separation: Python for logic, HTML for UI
  • Easy updates: Modify design without touching backend code
  • Reusable layouts: Common structure across pages
  • Jinja2 support: Dynamic content and logic in templates
